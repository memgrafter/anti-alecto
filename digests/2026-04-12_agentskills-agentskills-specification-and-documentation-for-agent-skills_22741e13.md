### TL;DR
Agent Skills is an open format, maintained by Anthropic, for packaging instructions, scripts, and resources into reusable “skills” that agents can discover and apply to specific tasks.

### Key Quote
“Skills are folders of instructions, scripts, and resources that agents can discover and use to perform better at specific tasks.”

### Summary
- **What it is**
  - A GitHub repository for **Agent Skills**, described as “a simple, open format for giving agents new capabilities and expertise.”
  - Skills are organized as **folders** containing:
    - instructions
    - scripts
    - resources
  - The intent is **write once, use everywhere** for agent capabilities.

- **What this repo contains**
  - The **specification**
  - The **documentation**
  - A **reference SDK**
  - A pointer to **example skills** in the Anthropic skills repo

- **Getting started / key links**
  - **Documentation:** https://agentskills.io
    - Guides and tutorials
  - **Specification:** https://agentskills.io/specification
    - Format details
  - **Example Skills:** https://github.com/anthropics/skills
    - Shows what skills can look like in practice
  - **Discord:** https://discord.gg/MKPE9g8aUy
    - Community discussion

- **Governance / ownership**
  - The format is **maintained by Anthropic**
  - It is explicitly **open to community contributions**
