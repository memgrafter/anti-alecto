### TL;DR
A Reddit user compiled a practical masterlist of AI chat platforms that support either BYOK API keys or local models, split into frontend-only and all-in-one apps, with notes on platform support, beta status, and provider limitations.

### Key Quote
“**This list is designed for proxy users who either bring their own API key or rely on local models.**”

### Summary
- **What this is:** A resource post in **r/aichapp** listing platforms for people who want to use **their own API keys (BYOK)** or run **local models**.
- **Definitions given in the post:**
  - **BYOK:** You provide your own paid API key from providers like OpenAI, Claude, DeepSeek, OpenRouter, etc.
  - **Local (Native):** The model runs directly on the device.
  - **Local (Network):** The app connects to another device on the same network running the model backend.
  - **Frontend UI:** Only a client; needs an API or local backend.
  - **All-in-One:** Includes its own local model backend, sometimes with BYOK too.
- **Frontend UI category includes:**
  - **Agnaistic** — BYOK + local network, web
  - **Chattica** — BYOK + local network, Android / iOS coming soon; selfhost only via LM Studio and KoboldCPP
  - **Chub AI** — BYOK, web / Android APK / iOS coming soon
  - **Duren AI** — BYOK, web
  - **Hakawati** — BYOK + local network, desktop
  - **Janitor AI** — BYOK, web / mobile coming soon
  - **Lettuce AI** — BYOK, Android APK, currently beta
  - **Loreblendr** — BYOK + local network, iOS, beta; supports MCP plugin
  - **MegaNova Chat** — BYOK, web, beta
  - **NeoTavern** — local network, desktop / Android; rewritten frontend for SillyTavern
  - **OMate Chat** — BYOK, Android / iOS
  - **Risu AI** — BYOK + local network, web / desktop
  - **Saucepan AI** — BYOK, web; provider support limited to OpenAI, DeepSeek, Anthropic, and Chutes
  - **School House AI** — BYOK, desktop; Android coming soon; beta
  - **SillyTavern** — BYOK + local network, desktop / Android via Termux
  - **TavernAI** — local network only, older/deprecated legacy app
  - **Tavo** — BYOK, Android APK / Play Store / iOS
  - **WyvernChat** — BYOK, web, beta
- **All-in-One category includes:**
  - **Backyard AI** — local native only in the legacy desktop app; desktop app is deprecated
  - **ChatterUI** — BYOK + local native + local network, Android APK
  - **Hammer AI** — BYOK + local native, desktop / web; web also supports local models via WebGPU
  - **Layla** — local native only, APK lite version free; Android and iOS paid apps
  - **Maid** — BYOK + local native, Android
  - **PocketPal** — local native only, Android / iOS
- **Important warning about Backyard AI:**
  - Although it has Android, iOS, and web versions, those rely on Backyard’s proprietary cloud.
  - They **do not** support BYOK or local model running on phone hardware.
  - Only the **legacy desktop app** supports local models.
