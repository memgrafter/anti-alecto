### TL;DR
Ronak Malde argues that current AI agent harnesses are too human-designed and highlights a paper proposing that an LLM itself should execute a natural-language SOP for orchestrating subagents, memory, and compaction.

### Key Quote
“Given a natural language SOP of how an agent should orchestrate subagents, memory, compaction, etc., we can just have an LLM execute that logic!”

### Summary
- This is a short X post by **Ronak Malde (@rronak_)** dated **Mar 29, 2026**.
- Core thesis: **traditional agent harnesses are too restrictive** because they are explicitly designed in code by humans.
- He points to a **new paper from Tsinghua and Shenzhen** suggesting a different approach:
  - Instead of hardcoding the harness, **let AI run the harness itself**.
