### TL;DR
“Overview - Agent Skills” explains an open standard originally developed by Anthropic for packaging agents’ instructions, scripts, and resources into reusable skill folders that add task-specific context and capabilities on demand.

### Key Quote
“Skills solve this by giving agents access to procedural knowledge and company-, team-, and user-specific context they can load on demand.”

### Summary
- **What Agent Skills are**
  - Folders containing **instructions, scripts, and resources** that agents can discover and use.
  - Designed to make agents do work **more accurately and efficiently** by supplying missing context.
  - Skills let agents load **procedural knowledge** and **company-, team-, and user-specific context** when needed.

- **Who they are for**
  - **Skill authors**: build capabilities once and deploy them across multiple agent products.
  - **Compatible agents**: gain new capabilities for end users “out of the box.”
  - **Teams and enterprises**: capture organizational knowledge in **portable, version-controlled packages**.

- **What skills can enable**
  - **Domain expertise**: reusable instructions for workflows like legal review or data analysis.
  - **New capabilities**: examples include creating presentations, building MCP servers, and analyzing datasets.
  - **Repeatable workflows**: multi-step tasks become consistent and auditable.
  - **Interoperability**: the same skill can be reused across different skills-compatible agent products.
