### TL;DR
Former GitHub CEO Thomas Dohmke’s new startup, Entire, raised a record $60 million seed at a $300 million valuation to build infrastructure for managing AI-generated code, starting with an open source product called Checkpoints.

### Key Quote
“We are living through an agent boom, and now massive volumes of code are being generated faster than any human could reasonably understand.”

### Summary
- **What happened**
  - TechCrunch reports that **Thomas Dohmke**, former CEO of **GitHub**, raised **$60 million** in seed funding for his startup **Entire**.
  - The round values Entire at **$300 million**.
  - TechCrunch says this is the **largest-ever seed round for a dev tool startup**, according to lead investor **Felicis**.

- **What Entire does**
  - Entire is building tools for developers to **manage code written by AI agents**.
  - Its product vision has **three components**:
    - a **Git-compatible database** to unify AI-produced code
    - a **“universal semantic reasoning layer”** for coordination among multiple AI agents
    - an **AI-native user interface** designed for human-agent collaboration

- **First product: Checkpoints**
  - The first release is an **open source tool** called **Checkpoints**.
  - It automatically links each software submission from an agent with the **context that created it**, including:
    - prompts
    - transcripts
  - The goal is to let developers **review, search, and learn from** why the AI produced the code.

- **Why this matters**
  - The article frames Entire as a response to the growing flood of code generated by AI coding agents.
  - It notes that popular open source projects are increasingly overwhelmed by suggested contributions, some of which may be **“AI slop”** — poorly designed or unusable code.

- **Founder background**
  - Dohmke was CEO of **Microsoft’s GitHub for four years**.
  - He left in **August 2025** to found a startup.
  - During his tenure, he oversaw the rise of **GitHub Copilot**.

- **Investors**
  - Other investors named in the seed round include:
    - **Madrona**
    - **M12**
    - **Basis Set**
    - **Harry Stebbings**
    - **Jerry Yang**
    - **Olivier Pomel** (Datadog founder and CEO)
