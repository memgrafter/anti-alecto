### TL;DR
Entire is an open-source CLI that records AI agent sessions and links them to git commits, turning each commit into a searchable checkpoint of both code and the reasoning behind it.

### Key Quote
"Every commit gets a checkpoint: the code change paired with the full agent session that produced it."

### Summary
- **What it is:** Entire is a CLI tool for git-based projects that captures AI agent sessions on every push/commit and indexes them alongside commits.
- **Core promise:** It preserves the “why” behind code changes, not just the final diff, so you can later search how and why something was built.
- **How it works:**
  - Hooks into your existing git workflow.
  - Captures conversations/sessions from supported AI coding tools.
  - Stores checkpoints directly in your git history.
- **Supported tools mentioned:** Claude Code, Gemini CLI, OpenCode, Cursor, GitHub Copilot CLI, and FactoryAI.
- **Installation:** The page promotes a one-line install:
  - `curl -fsSL https://entire.io/install.sh | bash`
- **Key benefits claimed:**
  - No context switching or new UI.
  - Works with any git-based project.
  - Open source and MIT licensed.
  - No hosted service or external database.
  - Lets teams search, view, and share the intent behind code changes.
  - Helps agents avoid “starting from zero” by giving them history of past decisions.
- **Main positioning:** Entire frames itself as “context attached, not archived,” meaning the context stays with the repo instead of being sent elsewhere.
