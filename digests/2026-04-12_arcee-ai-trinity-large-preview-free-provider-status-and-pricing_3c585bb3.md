### TL;DR
Arcee’s Trinity-Large-Preview is a 400B sparse MoE open-weight model aimed at creative, conversational, and agentic use cases, with 13B active parameters per token and a preview API currently running at 128k context via 8-bit quantization.

### Key Quote
“the Preview API currently served at 128k context using 8-bit quantization for practical deployment.”

### Summary
- **Model identity:** Trinity-Large-Preview is described as a frontier-scale open-weight language model from Arcee.
- **Architecture:**  
  - 400B-parameter sparse Mixture-of-Experts model  
  - 13B active parameters per token  
  - Uses **4-of-256 expert routing**
- **Strengths / intended use cases:**  
  - Creative writing  
  - Storytelling  
  - Role-play  
  - Chat scenarios  
  - Real-time voice assistance
- **Agentic/tool use:**  
  - Marketed as stronger than a typical reasoning model in these interactive settings  
  - Trained to work well with agent harnesses such as **OpenCode**, **Cline**, and **Kilo Code**  
  - Designed to handle complex toolchains and long, constraint-heavy prompts
- **Context length:**  
  - Native support for up to **512k tokens**  
  - The **Preview API** is currently served at **128k context**
- **Deployment details:**  
  - Uses **8-bit quantization** for practical deployment
- **Positioning:**  
  - Presented as an **efficiency-first** model  
  - Emphasizes **production-oriented frontier performance**, **open weights**, and **permissive licensing** for real-world use and experimentation
