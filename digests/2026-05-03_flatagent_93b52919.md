### TL;DR
`flatagent.py` implements the FlatMachines adapter that wraps a `flatagents.flatagent.FlatAgent` into the platform’s `AgentExecutor` interface, translating responses, usage/cost, errors, rate limits, tool calls, and metadata into FlatMachines’ normalized result format.

### Key Quote
"FlatAgent adapter for FlatMachines."

### Summary
- **What this file is**
  - A Python adapter layer between **FlatMachines** and the external **flatagents** package.
  - Defines:
    - `_map_error_code(...)`
    - `FlatAgentExecutor`
    - `FlatAgentAdapter`

- **Dependency behavior**
  - Imports `FlatAgent` and profile helpers from `flatagents`.
  - If `flatagents` is missing, it raises:
    - `ImportError("flatagents is required for FlatAgentAdapter")`

- **Error mapping logic**
  - `_map_error_code(error_type, status_code)` normalizes errors into FlatMachines codes:
    - `429` → `rate_limit`
    - `401` / `403` → `auth_error`
    - `400` → `invalid_request`
    - `5xx` → `server_error`
    - error-name heuristics:
      - contains `ratelimit` / `rate_limit` → `rate_limit`
      - contains `timeout` → `timeout`
      - contains both `content` and `filter` → `content_filter`
      - contains both `context` and `length` → `context_length`
    - default → `server_error`

- **`FlatAgentExecutor` responsibilities**
  - Wraps a `FlatAgent` instance and exposes FlatMachines executor methods.
  - `metadata` property delegates to `self._agent.metadata` if present.
  - `_sync_stream_callback()` propagates `_stream_event_callback` into the underlying runtime executor if it exists.
  - `cancel()`:
    - looks for `self._agent._runtime_executor.cancel`
    - returns `False` if unavailable
    - otherwise awaits the cancel function

- **Response mapping**
  - `_map_response(response, delta_calls, delta_cost)` converts a `FlatAgent` response into a FlatMachines `AgentResult`.
  - Maps:
    - **usage**
      - input/output/total tokens
      - cache read/write tokens
      - `api_calls` from `delta_calls`
    - **cost**
      - input/output/cache read/cache write/total
      - or `total` from `delta_cost` if detailed cost is missing
    - **error**
      - code from `_map_error_code`
      - type, message, retryable
      - includes `status_code` when available
    - **rate limit**
      - starts from `build_rate_limit_state(raw_headers, retry_after)`
      - fills in `windows` from normalized request/token limits if needed
      - sets `limited` using `response.rate_limit.is_limited()` if not already true
    - **provider_data**
      - passes through existing provider data
      - injects adapter-level `provider` and `model` if missing
      - includes raw headers if available
      - removes `None` values
    - **finish_reason**
      - uses `response.finish_reason.value`
      - or `"error"` if there is an error and no explicit finish reason
    - **tool_calls**
      - converts tool call objects into plain dicts:
        - `id`, `name` (from `tc.tool`), `arguments`
    - **metadata**
      - merges agent metadata and response metadata, with response metadata updating agent metadata
    - also preserves:
      - `output`
      - `content`
      - raw response
      - `rendered_user_prompt`

- **Execution methods**
  - `execute(...)`:
    - records pre-call totals for API calls and cost
    - syncs stream callback
    - passes `context` and optional `session_id` to `self._agent.call(...)`
    - computes deltas in API calls and cost
    - returns mapped `AgentResult`
  - `execute_with_tools(...)`:
    - same pattern as `execute`
    - additionally passes `tools` and `messages` to `self._agent.call(...)`

- **Adapter creation**
  - `FlatAgentAdapter.type_name = "flatagent"`
  - `create_executor(...)`:
    - discovers profiles file using:
      - `discover_profiles_file(context.config_dir, context.profiles_file)`
    - loads profiles from file if present
    - merges with `context.profiles_dict` via `resolve_profiles_with_fallback(...)`
    - creates a `FlatAgentExecutor` in one of two ways:
      - if `agent_ref.ref` exists:
        - resolves relative path against `context.config_dir`
        - constructs `FlatAgent(config_file=..., profiles_dict=...)`
      - if `agent_ref.config` exists:
        - constructs `FlatAgent(config_dict=..., config_dir=context.config_dir, profiles_dict=...)`
      - otherwise raises:
        - `ValueError(f"FlatAgent reference missing ref/config for agent '{agent_name}'")`
  - `_resolve_ref(ref, context)`:
    - returns absolute paths unchanged
    - otherwise joins `ref` with `context.config_dir`
