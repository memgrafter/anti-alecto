### TL;DR
Obsidian Web Clipper is a free, open-source browser extension that saves web content directly into an Obsidian vault as local Markdown, with highlighting, templates, hotkeys, and customizable data extraction.

### Key Quote
"Your clips are saved to durable Markdown files."

### Summary
- **What it is:** A browser-based web clipping tool for Obsidian that lets you highlight and capture web pages with one click.
- **Core workflow:**
  - **Capture** web pages into your vault.
  - Use **templates** to control how saved content is formatted.
  - **Highlight** passages so important text stays visible when revisiting the page.
- **Privacy and portability:**
  - Content is stored **locally** in your Obsidian vault.
  - It is **open source** and designed to avoid **lock-in**.
  - Clips are saved as **Markdown files**.
  - Highlights and settings can be exported to **JSON**.
- **Productivity features:**
  - **Hotkeys** for faster clipping.
  - **Auto-apply templates** based on the source website.
- **Extraction capabilities:**
  - Can capture page metadata like **meta tags** and **Schema.org variables**.
  - Supports extracting content via **element selectors**.
- **Data manipulation:**
  - Includes templating tools to **modify page data before saving** it to the vault.
- **Positioning:** Presented as a privacy-respecting, flexible clipping tool that turns the browser into a better capture interface for Obsidian users.
