### TL;DR
Nia is an indexing and retrieval layer for AI agents that can ingest many source types, let agents retrieve/traverse/cite them through one API, and claims to be faster, cheaper, and more reliable than alternatives.

### Key Quote
"Point Nia at any source — code, docs, PDFs, datasets, the web — and give every agent one API to retrieve, traverse, and cite."

### Summary
- **What Nia is**
  - A product from trynia.ai positioned as “Context for AI Agents.”
  - Described as “the index and search layer for AI agents.”
  - Designed to let agents access “ground truth” reliably.

- **Core promise**
  - Users can point Nia at many content types:
    - code
    - docs
    - PDFs
    - datasets
    - the web
  - Nia then provides a single API for agents to:
    - retrieve information
    - traverse sources
    - cite sources
  - The site claims the system is:
    - scalable
    - “5x cheaper”
    - reliable

- **Benchmark / positioning**
  - The page says retrieval is “the hardest part of agent reliability.”
  - It claims Nia was tested on “the worst case”:
    - bleeding-edge APIs
    - LLMs have never seen them before
  - It claims Nia “beat every alternative.”
  - A linked piece of supporting material is referenced as “Read the full research →”
  - The site emphasizes that “the same engine works on any source.”

- **Examples / use cases shown**
  - **Cursor Forum Search**
    - Search feature discussions, troubleshooting threads, and community knowledge from Cursor’s forum.
  - **HuggingFace Datasets**
    - Visualize, index, and search datasets, including anime quotes and scientific papers.
  - **Paul Graham Essays**
    - Ask questions grounded in 120+ essays about startups, writing, technology, and life.
  - **Simon Willison’s Weblog**
    - Index and search 4,500+ blog posts about Datasette, SQLite, LLMs, and the modern web.
  - Mentions additional developer uses:
    - code review agents that fetch up-to-date documentation on demand
    - law research agents indexing legal PDFs from Florida, California, and other states

- **Pricing tiers**
  - **Free**
    - 3 lifetime indexes
    - 50 queries/month
    - 20 web searches/month
    - 5 contexts
  - **Builder**
    - includes Free features plus:
      - 50 indexing/month
      - 5 concurrent indexes
      - 1,000 queries/month
      - 30 deep research/month
      - 200 web searches/month
      - 100 contexts
      - 30 Oracle/month
      - 30 Tracer/month (3 concurrent)
  - **Team**
    - includes Builder features plus:
      - 500 indexing/month
      - 50 concurrent indexes
      - 5,000 queries/month
      - 200 deep research/month
      - 1,000 web searches/month
      - 1,000 contexts
      - 200 Oracle/month
      - 200 Tracer/month (10 concurrent)
  - **Business**
    - unlimited indexing, concurrent indexes, queries, deep research, web searches, contexts, Oracle
    - unlimited Tracer (50 concurrent)
    - dedicated support
  - **Enterprise**
    - everything unlimited
    - unlimited Tracer (100 concurrent)
    - SOC 2 compliance
    - SLA
    - dedicated onboarding/support
    - self-hosted deployment available
