### TL;DR
OrbStack’s pricing page says the app is free for personal/non-commercial use, while business/commercial use costs $8 per user per month billed annually, with higher-tier features like Debug Shell, priority support, SAML SSO, and invoice/PO payments.

### Key Quote
“Free to try, no card required”

### Summary
- **Personal, non-commercial use: $0**
  - OrbStack is **always free for personal use**.
  - Includes:
    - Running containers
    - Running Linux machines
    - Community support

- **Business & commercial use: $8 per user per month**
  - Billed as **$96 annually**.
  - Includes:
    - Commercial use
    - Debug Shell for easier container debugging
    - Priority support
    - Batch organization purchases

- **Companies with advanced needs**
  - Additional enterprise-oriented features listed:
    - SAML SSO
    - Invoice and PO payments
    - “More coming soon”

- **Trial / purchase note**
  - The page emphasizes that it is **free to try, no card required**.
