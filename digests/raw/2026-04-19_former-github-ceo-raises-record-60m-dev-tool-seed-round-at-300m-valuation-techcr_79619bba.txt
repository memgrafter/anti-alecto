# Former GitHub CEO raises record $60M dev tool seed round at $300M valuation

[Julie Bort](https://techcrunch.com/author/julie-bort/)

6:30 AM PST · February 10, 2026

[](https://www.facebook.com/sharer.php?u=https%3A%2F%2Ftechcrunch.com%2F2026%2F02%2F10%2Fformer-github-ceo-raises-record-60m-dev-tool-seed-round-at-300m-valuation%2F)[](https://twitter.com/intent/tweet?url=https%3A%2F%2Ftechcrunch.com%2F2026%2F02%2F10%2Fformer-github-ceo-raises-record-60m-dev-tool-seed-round-at-300m-valuation%2F&text=Former+GitHub+CEO+raises+record+%2460M+dev+tool+seed+round+at+%24300M+valuation&via=techcrunch)[](https://www.linkedin.com/shareArticle?url=https%3A%2F%2Ftechcrunch.com%2F2026%2F02%2F10%2Fformer-github-ceo-raises-record-60m-dev-tool-seed-round-at-300m-valuation%2F&title=Former+GitHub+CEO+raises+record+%2460M+dev+tool+seed+round+at+%24300M+valuation&summary=Thomas+Dohmke%27s+new+startup+offers+an+AI+system+to+allow+developers+to+better+manage+all+the+code+AI+agents+produce.&mini=1&source=TechCrunch)[](https://www.reddit.com/submit?url=https%3A%2F%2Ftechcrunch.com%2F2026%2F02%2F10%2Fformer-github-ceo-raises-record-60m-dev-tool-seed-round-at-300m-valuation%2F&title=Former+GitHub+CEO+raises+record+%2460M+dev+tool+seed+round+at+%24300M+valuation)[](mailto:?subject=Former+GitHub+CEO+raises+record+%2460M+dev+tool+seed+round+at+%24300M+valuation&body=Article%3A+https%3A%2F%2Ftechcrunch.com%2F2026%2F02%2F10%2Fformer-github-ceo-raises-record-60m-dev-tool-seed-round-at-300m-valuation%2F)[](https://techcrunch.com/2026/02/10/former-github-ceo-raises-record-60m-dev-tool-seed-round-at-300m-valuation/)

Former GitHub CEO Thomas Dohmke has raised the largest-ever seed round for a dev tool startup, according to its lead backer, Felicis. The startup, [Entire](http://entire.io/), has raised $60 million at a $300 million valuation.

Entire offers an open source tool to help developers better manage code written by AI agents.

Entire’s tech has three components. One is a Git-compatible database to unify the AI-produced code. Git is a distributed version control system popular with enterprises and used by open source sites like GitHub and GitLab.

Another component is what it calls “a universal semantic reasoning layer” intended to allow multiple AI agents to work together. The final piece is an AI-native user interface designed with agent-to-human collaboration in mind.

The first product Entire is releasing is an open source tool it calls Checkpoints that automatically pairs every bit of software the agent submits for use in a software project with the context that created it, including prompts and transcripts. The idea is to allow the human developer to review, search, and perhaps even learn from why the AI did what it did.

Entire hopes to help developers better deal with the large volumes of software created by AI coding agents. Popular open source projects are [particularly overwhelmed these days with suggested code contributions](https://www.infoworld.com/article/4129056/is-ai-killing-open-source.html) that may or may not be AI slop — meaning poorly designed and possibly unusable code.

Dohmke explains in the press release: “We are living through an agent boom, and now massive volumes of code are being generated faster than any human could reasonably understand. The truth is, our manual system of software production — from issues, to git repositories, to pull requests, to deployment — was never designed for the era of AI in the first place.”
