This document provides a high-level introduction to the Aider codebase architecture, explaining the primary components, their interactions, and the overall system design. Aider is an AI pair programming tool that integrates Large Language Models (LLMs) with version control and code editing workflows to assist developers in modifying existing codebases.
For detailed information about specific subsystems:
- Core architecture components: see Core Architecture
- Edit strategies and code modification: see Edit Strategies and Code Modification
- Repository understanding: see Repository Understanding and Context
- User interaction modes: see User Interaction Modes
- Model configuration: see Model Configuration and Capabilities
Aider is a command-line AI coding assistant that enables developers to collaborate with LLMs to edit code within their existing repositories. Users can request changes through natural language, and Aider applies those changes using various edit formats optimized for different LLMs. The system automatically manages git commits, provides repository context to the LLM, and supports over 130 programming languages.
