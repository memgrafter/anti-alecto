### TL;DR
Terminal Trove’s “TUI Terminal Tools” page is a large curated directory of terminal-based text user interface apps, covering everything from file managers and database clients to AI assistants, monitoring tools, and chat clients.

### Key Quote
“Below is a list of terminal tools that are TUIs.”

### Summary
- This is a **category page / directory** on Terminal Trove, not a single tool review.
- It begins with a short primer explaining what **Text User Interfaces (TUIs)** are:
  - text-based layouts instead of GUIs
  - historically associated with early computing
  - still useful in resource-constrained environments and for tasks like server management, system administration, programming, and file management
- The main content is a **very long enumerated list** of TUI-oriented terminal tools, each with:
  - the tool name as a link
  - a one-line description of what it does
- The directory spans many tool types, including:
  - **File/navigation tools**: `yazi`, `ranger`, `nnn`, `lf`, `xplr`, `broot`, `superfile`
  - **Git/dev workflow tools**: `lazygit`, `gitui`, `gh-dash`, `forgit`, `diffnav`, `git-interactive-rebase-tool`
  - **System/resource monitors**: `htop`, `btop`, `glances`, `atop`, `nvtop`, `bottom`, `ctop`
  - **Logs/observability**: `lnav`, `lazyjournal`, `loggo`, `logshark`, `toolong`, `tailspin`
  - **Databases/data viewers**: `lazysql`, `rainfrog`, `db lab`, `squall`, `tabiew`, `csvlens`, `jq`-style viewers
  - **Messaging/social clients**: `slack-term`, `gomuks`, `aerc`, `neomutt`, `tut`, `toot`, `endcord`
  - **Cloud/infra tools**: AWS, Kubernetes, Docker, SSH, systemd, WireGuard, ECS, Proxmox, LDAP, Kafka, Redis, etc.
  - **LLM/AI tools**: `aider`, `claude-code`, `codex`, `gptop`, `tenere`, `oterm`, `parllama`, `yai`
  - **Editors/writers/readers**: `helix`, `vim`, `micro`, `amp`, `wordgrinder`, `presenterm`, `mdp`
  - **Network/security/debugging**: `mitmproxy`, `termshark`, `trippy`, `nethogs`, `strace-tui`, `inspect-cert-chain`, `ssl-checker`
- The list is broad and includes both:
  - well-known established projects, and
  - newer or niche tools with playful or experimental descriptions
- The page ends with a **call to action** encouraging users to post additional tools, plus Terminal Trove signup/social/footer navigation.
