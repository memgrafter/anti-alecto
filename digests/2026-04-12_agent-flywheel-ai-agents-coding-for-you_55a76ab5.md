Landing page by Jeffrey Emanuel pitching a “Flywheel” setup for AI-agent coding on a VPS, with a costed stack, onboarding wizard, and a planning-first workflow centered on decomposing projects into “beads.”

### Key Quote

### Summary
- This is a promotional/landing page for **Agent Flywheel**, billed as “AI Agents Coding For You.”
- The page is partially **scrape-corrupted**: several phrases and tool names are missing or garbled, so some specifics are unreadable rather than confidently recoverable.
- Core pitch:
  - Set up a **fully configured coding environment** with AI agents and modern developer tools.
  - The methodology is a **planning-first agentic development** process:
    - decompose projects into **beads**
    - detect convergence
    - coordinate **agent swarms**
    - ship faster
- “The Core Flywheel” section says beginners should start with just **three tools — Agent Mail, beads, and bv**.
- “Everything You Need” claims:
  - **One-liner install**
  - **Three AI agents** preconfigured for coding
  - **Idempotent & Safe** setup that can be rerun and resume on failure
  - **Vibe Mode** with passwordless access and dangerous flags enabled for throwaway environments
  - **Modern Shell** and an **interactive tutorial** via `onboard`
- “The Coding” section claims there are **twenty interconnected tools** for multi-agent workflows, but many tool names are not legible in the scrape.
- “13 Steps to Liftoff” suggests the wizard takes you from “I have a laptop” to “AI are coding for me” (also garbled).
- The page argues for using a **VPS** instead of a laptop or major cloud providers:
  - Laptops: too little RAM/CPU; agents use about **2GB RAM each**
  - For **10+ agents**, the page says you need **48–64GB RAM**
  - Not AWS/GCP/Azure: claims those are more expensive and less predictable
  - A **64GB VPS** is described as costing about **$40–56/month flat**
  - Benefit: runs **24/7**, so agents can work overnight
- “Is This For You?” says it is for people who:
  - want AI to write **real production code**
  - need **full control and complexity**
  - are willing to spend around **$500/month** on AI subscriptions
  - can follow step-by-step instructions, even without coding experience
- It is **not** for people who want:
  - a completely free solution
  - occasional snippet help only
  - mobile-first development
  - enterprise compliance out of the box
- Cost breakdown given:
  - **Cloud VPS**: 64GB RAM Ubuntu server (Contabo, OVH)
  - **Claude Max**: Claude Code CLI, cited at **$400 for power users (2 accounts)**
  - **ChatGPT Pro**: “ChatGPT 5.2 Pro for extended thinking planning”
  - Estimated monthly total: **$440–$656/month**
- Author identity and motivation:
  - **Jeffrey Emanuel** says he made this because many people asked him for help getting started with AI for software development.
  - He wanted one “from soup to nuts” resource for people with little computer expertise.
  - He also says the tools were originally built for his own consulting work with **Private Equity and Hedge Funds** and are now being shared freely.
