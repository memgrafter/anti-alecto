### TL;DR
Darkrealms is a long-running MS-DOS Renegade bulletin board from 1994 that serves as Fidonet Zone 1 Hub, hosts vintage computer files and Echomail archives, and exposes live network stats, downloadable nodelists, and instructions for getting listed in Zone 1.

### Key Quote
"Darkrealms is one of the last remaining bulletin boards running the MS-DOS Renegade software, online since 1994."

### Summary
- **What it is:** A classic bulletin board system (BBS) called **Darkrealms**, described as one of the last remaining systems running **MS-DOS Renegade**.
- **Since when:** Online since **1994**.
- **Role in Fidonet:** It is the **Fidonet Zone 1 Hub** and appears to function as a backbone/utility node for region-wide message routing and nodelist distribution.
- **Access methods:** Users can connect via **Telnet** or by **dialup modem** at **+1-647-847-2083**.
- **What it hosts:**
  - Massive archives of **Echomail**
  - **Hard-to-find vintage computer files**
  - Fidonet infrastructure files like nodelists and nodediffs
- **File listings shown:**
  - `ALLFILES.ZIP` — list of files on Darkrealms

  - `NODEDIFF.Z07`, `NODELIST.Z07`, `Z1DAILY.Z09`, `Z1DBINKD.ZIP`, `Z1BONE.NA`, `Z1USENET.ZIP`

    - Connections in/out

    - Data sent/received
    - Netmail and Echomail activity
    - Duplicate messages
- **Regional mapping:** Lists Fidonet **Region 10–19** with the covered U.S./Canada/other territories, including:
  - Region 12: Eastern Canada

- **Nodelisting requirements:** To be listed in Zone 1, users must:
  - Agree to **Policy 4**
  - Demonstrate a system that can exchange **Netmail**
  - Grab the latest **Nodelist or Dailylist**
  - Configure software correctly
  - Send a **Netmail** to the appropriate **Region or Network Coordinator**
