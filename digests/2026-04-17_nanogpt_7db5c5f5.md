### TL;DR
NanoGPT offers a two-tier AI subscription: a free pay-as-you-go plan and a $8/month Pro plan that bundles many open-source text models, some image models, API access, and a 5% discount on paid usage.

### Key Quote
“Private open-source AI for just $8/month”

### Summary
- **What this page is:** NanoGPT’s subscription/pricing page for access to its AI model platform.
- **Free tier / pay-as-you-go:**
  - Listed as **“ACTIVE”** and priced at **$0 per month**.
  - Includes:
    - Access to **every AI model there is**
    - Access to **image, video, text and voice**
    - Access to **web + API**
    - **Generous rate limits**
    - **One free model, available on web only**
- **Pro tier:**
  - Marked **“BEST VALUE”**
  - Costs **$8 per month**
  - Includes:
    - **Weekly included input tokens per week** (wording appears on the page, but the exact amount is not visible in the scrape)
    - **5% discount on all other text models**
    - **Both text and image models included**
    - **Both via web and API**
    - Everything in the free tier
- **Text models included in Pro:**
  - The page says **most open-source text models** are included, with examples:
    - **GLM 5** — described as a new flagship reasoning/chat model
    - **Kimi K2.5** — fast multimodal model for coding and agent tasks
    - **MiniMax M2.5** — productivity and tool-use workflows
    - **Qwen 3.5 397B** — large open-source MoE model with strong reasoning
    - **Uncensored Models** — Venice and more
    - **Roleplaying Models** — special finetunes
    - **Coding Specialists**
    - **Math Models**
    - “And dozens more open-source models”
  - Important caveat:
    - **GLM 5 is included**
    - **GLM 5.1 and GLM 5.1 Thinking are not currently included in Pro**
    - **TEE variants** and some unusually expensive open-source models are excluded
    - The page claims **99.9% availability** for popular models like GLM 5 and Kimi K2.5
    - Some roleplaying finetunes may depend on a single provider, so availability may vary
- **Image generation included in Pro:**
  - Models listed:
    - **HIDREAM**
    - **CHROMA**
    - **Z IMAGE TURBO**
    - **QWEN IMAGE**
  - The page says these premium image models are included at no extra cost, but also warns:
    - **Image models might not stay in the subscription forever**
    - It explicitly says not to base a subscription decision on image models alone
- **API access:**
  - Pro includes use via **website or API**
- **Paid model discount:**
  - While subscribed, users get **5% off all pay-as-you-go usage**
- **Privacy:**
  - The page emphasizes **“Privacy First”** and says **“Your data stays private”**
- **FAQ section:**
  - The page lists FAQ topics rather than answers in the scrape, including:
    - usage limits
    - multiple accounts
    - API and website use
    - API limits
    - support for **v1/completions**
    - context size limits
    - data privacy
    - whether paid models can still be used
    - which models get the 5% discount
    - what happens if quota is exceeded
    - whether a specific model is included
- **Referral program:**
  - Users can earn **5% from friends’ subscriptions**
  - Terms:
    - Share a personal link
    - When a friend subscribes or renews at **$8/mo**, the referrer earns **5% each time**
  - The page shows **“Generating your link…”**, suggesting the referral link feature is dynamic
