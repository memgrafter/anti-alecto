### TL;DR
Devin’s pricing page pitches its AI engineer for large-scale refactoring, then breaks down Core, Team, and Enterprise plans, centered on ACU-based usage, collaboration limits, and enterprise security/support features.

### Key Quote
“Devin's unit of work is an Agent Compute Unit, or ACU. It's a normalized measure of the computing resources Devin uses to complete a task, such as virtual machine time, model inference, and networking bandwidth.”

### Summary
- **Main pitch / case study**
  - The page opens with a Nubank migration story to illustrate Devin’s value on repetitive, high-volume engineering work.
  - Nubank reportedly used Devin on a major ETL refactor:
    - 8-year-old monolithic ETL
    - over 6 million lines of code
    - about 100,000 data class implementations
    - originally expected to take 18 months
  - Claimed results:
    - **8–12x efficiency gains**
    - **over 20x cost savings**
    - migrations completed in **weeks instead of months or years**
  - A quoted testimonial says Devin reduced engineering hours, was more stable, and required engineers mainly to review and merge PRs.
  - The page also claims:
    - fine-tuning doubled task completion scores
    - task speed improved 4x
    - roughly 40 minutes per sub-task dropped to 10 minutes
    - Devin built scripts/tools to automate repeated sub-steps
    - performance improved over time as it learned from more examples

- **Pricing tiers**
  - **Core**
    - “Pay as you go, starting at **$20**”
    - Includes:
      - Autonomous task completion
      - Advanced Capabilities
      - Devin IDE
      - Ask Devin
      - Devin Wiki
      - Devin API
      - Learns over time
    - Usage:
      - Unlimited users
      - Share and collaborate
      - Up to **10 concurrent Devin sessions**
      - No monthly commitment
      - Auto-reload settings for on-demand consumption
  - **Team**
    - **$500/month**
    - Everything in Core, plus:
      - Access to early feature releases and research previews
    - Usage:
      - Unlimited concurrent sessions
      - **250 ACUs included monthly**
    - Support:
      - Optional onboarding call with the Cognition team
  - **Enterprise**
    - **Custom pricing**
    - Everything in Team, plus:
      - Access to Devin Enterprise
    - Security:
      - Deploy in your **VPC**
      - **SAML/OIDC SSO**
      - Centralized enterprise admin controls
      - Teamspace isolation
    - Support:
      - Dedicated account team
      - Custom terms
      - Centralized billing and usage analytics across multiple Devin organizations

- **What ACUs are**
  - ACU = **Agent Compute Unit**
  - Defined as a normalized measure of Devin’s compute usage, including:
    - virtual machine time
    - model inference
    - networking bandwidth
  - Charging details:
    - Core: pay-as-you-go at **$2.25 per ACU**, with auto-recharge options
    - Team: **250 ACUs at $2.00 per ACU**, with optional auto-recharge after credits are used
  - ACUs are consumed only when Devin is actively working or the VM is running
  - Consumption depends on:
    - task complexity
    - prompt quality/specificity
    - codebase size / number of files touched
    - session runtime
  - API use on Team/Enterprise:
    - no extra API fee
    - API sessions consume ACUs like normal sessions

- **Ownership / legal**
  - The page states that users own code generated by Devin
  - All inputs and outputs are treated as the user’s intellectual property
  - It points readers to the Terms of Service for plan-specific details

- **Support / contact**
  - Mentions support at **support@cognition.ai**
  - Core includes a dedicated Slack Connect support channel
  - Team includes optional onboarding
  - Enterprise includes dedicated account/engineering support and custom terms
